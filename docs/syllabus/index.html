    Course Overview & Syllabus

    Class Description

    The focus of this class is to introduce the role of High Performance Computing (HPC, aka supercomputing) in research. We will discuss the fields where HPC is used and provide concrete examples where we describe the strategies used to scale applications to hundreds of processors. Students will learn when to scale from their laptops to HPC, what challenges that introduces, and how to address those challenges with efficient HPC workflows. The MIT SuperCloud will be used for hands-on examples using C/C++, Julia, Matlab, and/or Python. We will also demonstrate applications using other computing resources on campus, such as the Satori and Engaging clusters. Students should bring an existing research problem/application that they would like to scale as a project.

    This is a blended course with asynchronous and in-person live components. Much of the lecture will be available before class in pre-recorded short videos and class time will be spent on hands-on activities and student research project work. Students taking the class for MIT credit are required to complete a short report and presentation on their project.

    Learning Objectives

    • Where is Supercomputing used? When? Why?
    • Know the differences between desktop/laptop and HPC system/supercomputers.
    • Understand how to identify Supercomputing Workflows, plan for how to scale up, and measure performance.

    Expectations

    Students are expected to attend scheduled class sessions and review the associated asynchronous material prior to class. All students are highly encouraged to participate in project activities and give a short presentation on the last day of class.

      • Introduction to Supercomputing

      • Getting Started with the MIT Supercloud
      • Print out or download reference guide
      • Review Glossary
    • Class Session:
      • Course Overview and Introductions
      • Q&A for Async material
      • Activity: Puzzle Lab
      • Introducing Workflow Case STudies
      • Introduce Projects
      • In-class project work
      • SuperCloud access troubleshooting
      • Start setting up your environment
      • Work toward running your code in serial

    Day 2: Serial Optimization and Parallel Speedup

    • Asynchronous Preparation:
      • Supercomputing Systems Deep Dive
      • Best Practices for Scalable Development
    • Class Session:
      • Q&A for Async material
      • Workflows Case Study Activity
      • Best Practices for Scalable Development
      • Predicting, measuring, and optimizing performance
      • In-Class Project Work
      • Identify workflow
      • Measure serial performance

    Day 3: Building and Running Parallel Workflows

    • Asynchronous Preparation
      • Batch vs Interactive Jobs
      • Using the Scheduler
      • Submitting Jobs
      • Running Throughput and Loosely Coupled Applications
    • Class Session:
      • Q&A for Async Material
      • Scheduler Activity
      • Scaling Up Case Studies
      • In-Class Project Work
      • Start building workflow
      • Select appropriate hardware

    Day 4: Distributed Computing

    • Asynchronous Preparation:
      • Types of Parallel Programs
      • PGAS Parallel Programs (Partitioned Global Address Space)
      • MPI
    • Class Session:
      • Q&A for Async Material
      • Distributed Example Walkthroughs
      • In-Class project work
      • Presentations or group discussions on projects
      • Presentations (aim for 5 minutes)
        • General, high level overview of application (for a general audience)
        • Proposed workflow
        • Current status
        • Challenges that you ran into
        • Scaling curve (either completed curve or planned approach for gathering)
